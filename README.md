# JS Playground

- To provide javascript playground to practice skills

### Requirement

- Node >= v10
- npm >= v5.2

### How to develop

- JavaScript

  - Start server

    ```
    $ npm start
    ```

- TypeScript

  - Start server

    ```
    $ npm run start:ts
    ```

  - Watch TS changes to update simultaneously

    ```
    $ npm run watch:ts
    ```

### How to apply changes

  - Simply refresh your browser
